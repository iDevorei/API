# API

## Preconditions

- Ссылка на продукт: [Megaplan](https://m74965812.megaplan.ru/)

- Здесь лежит [Документация](https://m74965812.megaplan.ru/api/v3/docs) на API

- Данные для авторизации см. в Классе.

### Scenario

- Создать коллекцию.

1. Авторизоваться под Cотрудником 1.
1. Создать задачу. Ответственный по задаче: Сотрудник 2.
1. Написать описание задачи. Текст любой.
1. Авторизоваться под Сотрудником 2.
1. Принять задачу, поставленную Сотрудником 1.
1. Завершить поставленную задачу.
1. Написать комментарий по задаче. Текст любой.
1. Авторизоваться под Сотрудником 1.
1. Принять завершённую задачу.
1. Удалить задачу.
