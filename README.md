## Preconditions

- Ссылка на продукт: [Megaplan](https://m74965812.megaplan.ru/)

- Здесь лежит [Документация](https://m74965812.megaplan.ru/api/v3/docs) на API

- Данные для авторизации см. в Классе.

### Scenario

- Создать коллекцию.

1. Авторизоваться под Cотрудником 1.
1. Создать задачу. Ответственный по задаче: Сотрудник 2.
1. Написать описание задачи. Текст любой.
1. Авторизоваться под Сотрудником 2.
1. Принять задачу, поставленную Сотрудником 1.
1. Завершить поставленную задачу.
1. Написать комментарий по задаче. Текст любой.
1. Авторизоваться под Сотрудником 1.
1. Принять завершённую задачу.
1. Удалить задачу.

### Scenario Part 2

- Добавить в свою коллекцию переменные, прокинуть их в методы и добавить скрипты, чтобы сетать в эти переменные значения

- В текущую коллекцию добавить тесты:

1. Авторизоваться под сотрудником 1.
    - Добавить тест на: код ответа

1. Создать задачу. Ответственный по задаче: Сотрудник 2.
    - Добавить тесты на: код ответа, название задачи, ответственного по задаче, статус задачи

1. Написать описание задачи. Текст любой
    - Добавить тесты на: код ответа, описание задачи

1. Авторизоваться под Сотрудником 2.
    - Добавить тесты на: код ответа

1. Принять задачу, поставленную Сотрудником 1.
Добавить тесты на: код ответа, статус задачи

1. Завершить поставленную задачу.
    - Добавить тесты на: код ответа, статус задачи

1. Написать комментарий по задаче. Текст любой
    - Добавить тесты на: код ответа, комментарий

1. Авторизоваться под Сотрудником 1
    - Добавить тест на: код ответа

1. Принять задачу
    - Добавить тесты на: код ответа, статус задачи

1. Удалить задачу
    - Добавить тесты на:  код ответа, наличие текста в теле ответа
